## 1. Install environment
```
    Install nodejs 18
    Install mysql
```

## 2. Run product

### Reset project
``` sh
    npm run clean
```

### Configuration environment (.env file) [ => Click to .env file](.env)
#

### Build project
``` sh
    npm run build
```

### Start project
``` sh
    npm start
```

### Install node_modules
``` sh
    npm install
```

## 3. Run development
### Clean project
``` sh
    npm run clean
```

### Install node_modules
``` sh
    npm install
```

### Run project
``` sh
    npm run dev
```

## 4. Api document (.env file) [ => Click to swagger.yml file](swagger.yml)
